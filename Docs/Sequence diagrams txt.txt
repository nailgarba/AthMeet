title Authentication


participant Application
participant withAuthenticator

participant Cognito User Pool

participant Cognito Identity Pool

participant AWS Services

// ->
// :


Application->withAuthenticator:Application is launched
withAuthenticator->Cognito User Pool:Users enter sign in information
Cognito User Pool->Cognito User Pool:Sign in information is checked and authenticated
withAuthenticator<-Cognito User Pool:Successful authentication sends tokens
withAuthenticator->Cognito Identity Pool:Tokens are exchanged for AWS Credentials 
withAuthenticator<-Cognito Identity Pool:Sends AWS credentials based on Identity pool user's access
Application<-withAuthenticator:Provides application with AWS Access Credentials
Application->AWS Services:Access AWS Services through received credentials
Application<-AWS Services:AWS provides services
